# HFhockey

Application React Native (Expo) pour la Ligue Magnus.

## 🚀 Comment lancer l'application

### Pré-requis
1. Avoir **Node.js** installé (ce qui est le cas sur votre machine).
2. Avoir l'application **Expo Go** sur votre téléphone (disponible sur App Store / Play Store).
   - [iOS (App Store)](https://apps.apple.com/app/expo-go/id982107779)
   - [Android (Play Store)](https://play.google.com/store/apps/details?id=host.exp.exponent)

### Lancement
1. Ouvrez un terminal dans ce dossier :
   ```bash
   cd c:\Users\Utilisateur\OneDrive\Bureau\hockey
   ```
2. Installez les dépendances (si ce n'est pas déjà fait) :
   ```bash
   npm install
   ```
3. Lancez le serveur de développement :
   ```bash
   npx expo start
   ```
4. Scannez le **QR Code** qui s'affiche dans le terminal avec votre téléphone :
   - **iPhone** : Utilisez l'appareil photo standard.
   - **Android** : Utilisez l'appli Expo Go.

## 🛠 État du projet
- **Navigation** : Prête (Onboarding -> Tabs).
- **Design** : Components "Liquid Retina" implémentés.
- **Données** : Simulées (Mock Data) pour les matchs et équipes.

## 📱 Note sur l'affichage
L'application est conçue pour être "Immersive". Sur Expo Go, assurez-vous de secouer votre téléphone pour ouvrir le menu dév et recharger si nécessaire.
